# Assignment Tracker

A simple and efficient web application to track your academic assignments, deadlines, and completion status.

## Features
* **Create:** Add new assignments with titles and due dates.
* **Track:** View all active assignments.
* **Update:** Mark assignments as completed.
* **Delete:** Remove tasks that are no longer needed.
